Information :

In this HelpingHand Users are categorized into 2 types: Donors
                                                        Receivers
  
  User can Login using his/her Username,phone number and also via Google & Social media accounts.

  Pages : Home  | Donate | Receive | Receive(list)| Chat | History | Information | AboutUs | Contact | Feedback |
  
  Home Page : At Middle of the HomePage, there are 2 options :
                                                                Donate  &  Receive.
  
  
  Receive : Opens a new page showing option for browsing Items : Location wise(State and District) 
            and Category wise.
            In case if there are no items in that district under that selected Category then,
            HelpingHand will be dispalying the items in that State under that Category.
            Browse by Categories.
            Each item is displayed in list format showing {title} When clicked on Search button.
    
  Categories :  Food, Clothes, Accessories, Footwear, Gadgets, Books, Sport Items and Others
  
  Chat : When receiver decide to claim a Item and if he/she confirms it then end to end chat is provided 
         between donor and receiver.
  
  Donate : A webpage showing options for uploading an Image of the Item , A small Description and
           Contact Details to collect.
           There will be options of naming his/her state,district and the collecting address.

  Feedback : Users can Send their Feedback through this.This will be available only after using HelpingHand.
